    歡迎來到世新大學2015 Android製作教學課程

    本次課程著重在網路程式設計、彈性化AdapterView以及前後端整合。本次課程需要您先觀看過去年的教學影片。若您熟知git與指令模式，可使用以下指令直接取得本次課程相關的專案程式碼與講義

    $ mkdir workshop2015
$ cd workshop2015
$ git clone https://github.com/silencecork/SHUWorkshop2015.git

    課程說明

    第一部份會說明Android如何與Server進行連線，呼叫Server的API並且剖析回傳的JSON，接下來在Android中以客製化的方式呈現出來。
    第二部分會說明如何使用Node.js建立簡單的API Server，並連接MongoDB，依照連入的需求從MongoDB中儲存、刪除、查詢資料。
    第三部分會做整合，讓自己的的Android App與自己寫的Server連接

    事前準備

    本次課程會使用到的工具如下：
    JDK
    Eclipse (不是Android Studio)
    ADT
    Node.js

    使用到第三方函式庫

    本課程中會使用到兩個Open Source的函式庫
    Volley
    Picasso
